<h2 align="center">Quorum-IT’s Backend Assignment</h2>

## Description

Quorum-IT’s Backend Assignment Project

## Project setup

- After cloning the project, use the development branch

- Copy .env file to project directory

## Compile and run the project

The project use Docker, only run

```bash
$ docker-compose up --build
```

But, if you need to run without Docker

```bash
$ npm i

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
## API Documentation with Swagger

- After run the project, browse to: [Swagger ](http://localhost:3000/api)

## Login with super administrator user

- When running the API, a main super administrator user is generated by default
- Email: admin@mail.com
- Password: Admin-1234

## Migrations

- The migrations running automatically when app starts

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
