<h2 align="center">Quorum-IT’s Backend Assignment</h2>

## Description

Quorum-IT’s Backend Assignment Project

## Project setup

Copy .env file to project directory

```bash
$ docker-compose up --build
```

## Compile and run the project

The project use Docker, only run

```bash
$ docker-compose up --build
```

But, if you need to run without Docker

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
